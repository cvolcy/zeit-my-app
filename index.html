Date|JS

A simple lambda in nodejs that return the informations about the current date and time of the server.

Example : 


    const data = {
        timestamp: Unix epoch timestamp in milliseconds,
        date: The current date in text,
        time: The current time in text
              formatted as (hh:mm:ss GMT+0000 (UTC),
        timezoneOffset: The timezone offset of the server's timezone,
        name: My name,
        email: My email
    };

GraphQL|JS

A lambda in nodejs that acts as a GraphQL server connected to a MongoDB database.
It serve two models (Videos and Books).

Example : 


    {
        videos(lang:"en") {
            title url
        }
    }
    // Output
    {
        "videos": [
            {
                "title": "The 7 Steps of Machine Learning (AI Adventures)",
                "url": "https://www.youtube.com/embed/nKW8Ndu7Mjw"
            },
            { ... },
            ...
        ]
    }
